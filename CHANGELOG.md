# Changelog

All notable changes to `laravel-expo-channel` will be documented in this file.

## 1.3.0 - 2023-03-12

### Added

- PHP 8.3 support
- Laravel 11 support

### Removed

- PHP 8.2 support
- Laravel 10 support

## 1.2.0 - 2022-04-27

### Added

- PHP 8.2 support
- Laravel 10 support

### Removed

- PHP 8.1 support
- Laravel 9 support

## 1.1.0 - 2022-09-03

### Changed

- `ext-zlib` (Gzip compression) is now optional.

## 1.0.0 - 2022-09-02

- Initial release
